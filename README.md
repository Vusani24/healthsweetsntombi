# Health Sweets Ntombi

Static business website for Health Sweets Ntombi, an independent APLGO distributor business.

## Run locally

```powershell
npm start
```

The server uses port `3200` by default. You can change it with:

```powershell
$env:PORT = "3201"; npm start
```

## Edit prices

Product prices are in `index.html` inside each product card.
